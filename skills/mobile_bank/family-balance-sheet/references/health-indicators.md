# 财务健康指标

## 指标1：资产负债率

```
资产负债率 = 总负债 / 总资产 × 100%
```

| 数值 | 评级 | 含义 |
|------|------|------|
| < 30% | 🟢 健康 | 负债可控，财务弹性充足 |
| 30%-60% | 🟡 关注 | 负债适中，需注意新增负债 |
| > 60% | 🔴 警告 | 负债偏重，建议优先降负债 |

**注意**：房贷通常占比较大，若剔除房贷后负债率 < 20%，仍视为健康。需同时展示含/不含房贷两个口径。

## 指标2：流动性比率

```
流动性比率 = 流动资产 / 短期负债
```

流动资产 = 储蓄卡活期余额（不含定期存款和理财持仓）

短期负债 = 信用卡当期待还 + 近 3 个月内到期的贷款还款

| 数值 | 评级 | 含义 |
|------|------|------|
| > 3.0 | 🟢 充裕 | 短期偿债能力强 |
| 1.5-3.0 | 🟡 适中 | 基本覆盖短期负债 |
| 1.0-1.5 | 🟡 偏紧 | 需关注现金流 |
| < 1.0 | 🔴 不足 | 短期偿债风险，建议立即补充流动资金 |

## 指标3：月供收入比

```
月供收入比 = 月总还款额 / 月收入估算 × 100%
```

月总还款额 = 所有贷款月供 + 信用卡最低还款额

月收入估算方法（优先级从高到低）：
1. 用户直接提供
2. 近 3 个月工资入账平均值（通过 `transaction_history` 查询）
3. 近 3 个月所有收入平均值

| 数值 | 评级 | 含义 |
|------|------|------|
| < 30% | 🟢 安全 | 还款压力小 |
| 30%-50% | 🟡 关注 | 还款压力适中 |
| > 50% | 🔴 警告 | 还款压力大，影响生活质量 |

## 指标4：资产配置集中度

衡量资产是否过度集中于单一类型。

```
集中度 = max(各类资产占比)
```

资产分类：活期存款、定期存款、理财产品

| 数值 | 评级 | 含义 |
|------|------|------|
| < 60% | 🟢 分散 | 配置较均衡 |
| 60%-80% | 🟡 偏集中 | 建议适当分散 |
| > 80% | 🔴 过度集中 | 风险集中，强烈建议分散配置 |

## 综合评级规则

| 🟢 数量 | 🟡 数量 | 🔴 数量 | 综合评级 |
|---------|---------|---------|---------|
| 4 | 0 | 0 | 🟢 财务状况优秀 |
| 3+ | ≤1 | 0 | 🟢 财务状况良好 |
| 2+ | ≤2 | 0 | 🟡 财务状况一般，有优化空间 |
| 任意 | 任意 | 1 | 🟡 存在风险点，建议关注 |
| 任意 | 任意 | ≥2 | 🔴 财务状况需要改善 |
