# 还款计算方法

## 等额本息

每月还款额固定。

```
月还款额 = 本金 × 月利率 × (1 + 月利率)^期数 / ((1 + 月利率)^期数 - 1)
总利息 = 月还款额 × 期数 - 本金
```

**特点**：每月还款额固定，前期利息占比高，适合收入稳定的用户。

## 等额本金

每月本金固定，利息逐月递减。

```
每月本金 = 总本金 / 期数
第 N 月利息 = (总本金 - 已还本金) × 月利率
第 N 月还款额 = 每月本金 + 第 N 月利息
总利息 = (期数 + 1) × 总本金 × 月利率 / 2
```

**特点**：首月还款最高，逐月递减，总利息较等额本息少，适合前期还款能力强的用户。

## 对比输出格式

```
贷款方案对比
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

贷款期限：24个月

```

## 提前还款节省计算

```
节省利息 = 剩余期限利息总和 - 提前还款违约金（如有）
```

提前还款时需提示用户：
- 是否有最低还款期限限制
- 违约金比例（通常为提前还款金额的 1%-3%）
