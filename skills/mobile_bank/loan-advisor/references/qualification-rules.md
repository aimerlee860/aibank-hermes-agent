# 贷款资质评估规则

## 核心指标：负债收入比（DTI）

```
DTI = 月总还款额 / 月收入估算
```

### 月收入估算方法

优先级从高到低：
1. 用户直接提供月收入
2. 查询工资卡近 3 个月入账（`transaction_history`，transaction_type=工资）取平均值
3. 所有储蓄卡近 3 个月总入账 ÷ 3（粗略估算）

### DTI 阈值判断

| DTI | 评估结论 | 建议 |
|-----|---------|------|
| < 30% | 资质优良 | 可申请，额度充裕 |
| 30%-50% | 资质一般 | 可申请，建议控制额度 |
| 50%-70% | 资质偏弱 | 建议降低贷款额度或延长期限 |
| > 70% | 不建议 | 负债过重，建议先还清部分贷款 |

## 各贷款类型特殊要求

### 消费贷
- 最低年龄：22 岁
- 额度：1万-30万
- 期限：6/12/24/36 个月
- 无需抵押

### 车贷
- 首付比例：≥ 30%
- 额度：车价 × 70%
- 期限：12/24/36/48/60 个月
- 需提供购车合同

### 经营贷
- 需营业执照（经营满 1 年）
- 额度：10万-500万
- 期限：6/12/24/36 个月
- 可能需要抵押物

### 房贷（提前还款）
- 已有房贷用户查询：`mortgage_balance_query`
- 还款计划查询：`mortgage_repayment_plan`
- 提前还款预约：`mortgage_prepayment_appointment`
- 注意：部分银行提前还款需满贷 1 年，可能有违约金

## 以贷还贷分析

当用户有多笔贷款时，评估是否值得合并：

1. 列出所有贷款的剩余本金、利率、剩余期限
2. 计算加权平均利率
3. 若新贷款利率 < 加权平均利率 × 0.9，建议合并
4. 需考虑提前还款违约金
