## 验证方式详述

### L1 — 基础（短信验证码）
- 6位数字，有效期 5 分钟
- 每日最多发送 5 次（同号码）
- 发送间隔 60 秒

### L2 — 增强（短信验证码 + 生物识别）
- 顺序：先短信验证码 → 再生物识别（指纹/人脸二选一）
- 连续 3 次生物识别失败 → 锁定 30 分钟，提示去柜台处理

### L3 — 最高（U盾 / 视频核身 / 柜台）
- **U盾验证**：插入 U盾 → 短信验证码 → U盾动态口令（6位，60秒有效）
- **视频核身**：接通银行远程核身人员，读身份证号 + 人脸比对
- **柜台**：携带身份证原件到营业网点

## 高风险告警触发条件

以下情况自动触发安全告警，须用户确认：

| 触发条件 | 告警等级 | 建议操作 |
|---------|---------|---------|
| 陌生设备登录（新设备首次） | 中 | 确认是否本人，否则解绑 |
| 异地登录（距上次登录距离 > 500km） | 高 | 确认是否本人，否则冻结 |
| 凌晨登录（01:00–06:00） | 低 | 提醒用户关注 |
| 同一账号 24 小时内多城市登录 | 高 | 疑似账户被盗，建议冻结 |
| 连续 5 次密码错误 | 高 | 账户自动锁定 30 分钟 |
| 短时间内多次修改密码 | 中 | 人工审核触发 |

## 账户冻结说明

**冻结类型**：
- **主动冻结**（用户自助）：通过 App/助手冻结，可随时自助解冻
- **风控冻结**（系统触发）：因异常交易被系统冻结，须联系客服解除

**主动冻结影响**：
- 不可转账、消费、ATM 取款
- 可查询余额和明细
- 不影响代扣（房贷、水电费自动扣款）

## U盾使用规范

1. U盾仅限本人使用，不得外借
2. 不在他人电脑上使用 U盾
3. 交易完成后立即拔出 U盾
4. 丢失后 **立即挂失**，挂失前发生的损失由持卡人承担
5. U盾挂失后无法撤销，须重新申请（工作日 3–5 个工作日到货）
