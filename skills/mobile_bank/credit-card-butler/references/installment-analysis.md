# 分期分析规则

## 分期方案对比

### 可选期数与参考费率

| 期数 | 每期费率 | 折算年化利率（IRR） |
|------|---------|-------------------|
| 3期  | 0.75%   | ~16.2% |
| 6期  | 0.70%   | ~15.1% |
| 12期 | 0.65%   | ~14.0% |
| 24期 | 0.60%   | ~12.9% |

> 注：银行标注的"手续费率"远低于实际年化利率，因为本金在逐月归还。

### 分期成本计算

```
每期还款额 = 分期本金 / 期数 + 分期本金 × 每期费率
总手续费 = 分期本金 × 每期费率 × 期数
总还款额 = 分期本金 + 总手续费
```

## 最低还款代价计算

最低还款额通常为账单金额的 10%。未还部分按日利率 0.05%（年化 18.25%）计息，且全额罚息（从消费日起算）。

```
示例：账单 ¥5,200，最低还款 ¥520
- 剩余 ¥4,680 按日利率 0.05% 计息
- 30天利息 ≈ ¥70.20
- 若下期仍最低还款，利息将滚入本金（复利）
```

**结论**：最低还款年化利率约 18.25%，远高于分期。除非极短期周转（< 1 周还清），否则分期更优。

## 多卡还款优先级

按以下优先级排序：
1. **已逾期的卡** — 立即还款，避免征信影响
2. **还款日最近的卡** — ≤ 3 天内到期优先
3. **利率最高的卡** — 减少利息损失
4. **额度使用率最高的卡** — 使用率 > 70% 影响征信评分

## 额度优化建议

| 额度使用率 | 状态 | 建议 |
|-----------|------|------|
| < 30% | 健康 | 正常使用 |
| 30%-70% | 注意 | 适当控制消费 |
| 70%-90% | 偏高 | 建议还款降低使用率 |
| > 90% | 危险 | 可能影响征信，建议立即还款或申请临时额度 |

临时额度申请建议：
- 使用率 > 80% 且有大额消费需求时
- 申请额度建议为预计消费金额的 120%
- 有效期选择覆盖消费+还款周期
