# 资产配置方案生成规则

## 闲置资金识别

1. 取所有储蓄卡活期余额总和
2. 减去动态缓冲额：
   - 默认缓冲：¥10,000
   - 若最近30天日均支出 > ¥3,000：缓冲 = 月均支出 × 2
3. 剩余金额 = 可配置资金

**调用路径**：`balance_query(account_type=储蓄卡)` → `transaction_history(time_range=最近30天)` → 计算日均支出

## 风险等级 → 产品配置比例

| 用户风险等级 | 货币/存款类 | 固收理财/纯债基金 | 固收+/混合基金 | 股票/权益类 |
|------------|------------|----------------|--------------|-----------|
| 保守型（R1） | 60%–80% | 20%–40% | 0% | 0% |
| 稳健型（R2） | 30%–50% | 40%–60% | 0%–10% | 0% |
| 平衡型（R3） | 20%–30% | 30%–40% | 20%–30% | 0%–10% |
| 进取型（R4） | 10%–20% | 20%–30% | 20%–30% | 20%–30% |
| 激进型（R5） | 5%–10% | 10%–20% | 20%–30% | 40%–60% |

## 三档方案模板

输出时提供3档让用户选择：

### 保守档（低风险保本优先）
- 50% 大额存单或定期存款（锁定收益）
- 30% R1–R2 固收理财（灵活）
- 20% 货币基金（备用）

### 平衡档（收益与流动性兼顾）
- 30% 定期存款（1年期）
- 40% R2–R3 固收+理财/纯债基金
- 20% R3–R4 偏债混合基金
- 10% 货币基金（备用）

### 进取档（追求中长期增值）
- 20% 定期存款（稳定底仓）
- 30% R3 固收+理财
- 30% R4 偏股混合基金/指数基金
- 10% 贵金属（黄金对冲）
- 10% 货币基金（备用）

## 约束规则

- 单一产品 ≤ 总配置金额 40%
- 同风险等级产品不超过 3 只
- 封闭期产品合计 ≤ 可配置资金 60%（保留流动性）
- 必须保留活期缓冲，不可全部投入封闭期产品
- R4+ 产品须附风险提示

## 输出格式

```
💰 可配置资金：¥XX万

📊 推荐配置方案（平衡档）
━━━━━━━━━━━━━━━━━━━━━━━━
产品          金额      占比   风险  预期年化  期限
XXX理财       ¥X万      30%    R2    3.5%     1年
XXX债券基金   ¥X万      40%    R3    4.0%     开放
XXX指数基金   ¥X万      20%    R4    -        开放
货币基金      ¥X万      10%    R1    2.0%     随时

⚠️ 风险提示：理财非存款，市场有风险，以上为建议方案，非投资承诺。
```

## 到期续投逻辑

1. 查询所有即将到期产品（≤30天）
2. 按原产品风险等级匹配续投产品
3. 若市场行情显著变化（如利率下行），建议调整期限策略
4. 优先续投同系列产品（避免重新风险评估流程）
