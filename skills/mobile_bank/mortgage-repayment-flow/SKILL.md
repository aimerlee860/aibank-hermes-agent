# 固定房贷还款流程

当用户请求"还房贷"或类似表述时，按以下固定流程执行：

## 流程步骤

### 第一步：查询贷款信息
调用 `loan_query` 获取房贷月供金额和贷款账号：
- 贷款账号: `LN20230101`
- 月供金额: ¥8,500

### 第二步：检查0112账户余额
调用 `balance_query` 查询活期账户0112的余额。

### 第三步：判断是否需要转账
- 如果 0112 余额 ≥ ¥8,500：直接还款，跳过转账步骤
- 如果 0112 余额 < ¥8,500：执行转账补充资金

### 第四步：从其他账户转账（如需要）
调用 `account_list` 获取储蓄账户列表，优先从以下账户转账：
- **首选**: 定期账户 `0115`（余额通常充足）
- **次选**: 主账户 `0113`

转账金额: 固定 ¥10,000（确保还款后有余额缓冲）

调用 `internal_transfer`:
```
from_account: "0115" (或其他有余额的储蓄卡)
to_account: "0112"
amount: 10000
```

### 第五步：执行房贷还款
调用 `loan_repayment`:
```
loan_account: "LN20230101"
from_account: "0112"
repayment_amount: 8500 (月供金额，从loan_query获取)
```

### 第六步：汇报结果
汇总显示：
- 转账情况（是否转账、金额、来源账户）
- 还款情况（金额、贷款余额变化）
- 各账户余额变动

## 用户账户配置

| 账户 | 类型 | 用途 |
|------|------|------|
| 0112 | 活期储蓄 | 还款主账户 |
| 0113 | 活期储蓄（主账户） | 备用资金来源 |
| 0115 | 定期储蓄 | 主要资金来源 |
| LN20230101 | 房贷 | 贷款账号 |

## 注意事项

1. 如果所有储蓄账户余额都不足，提示用户资金不足
2. 还款前确认月供金额（可能变动）
3. 转账固定10,000元，确保还款后留有缓冲余额
4. 汇报时显示房贷剩余本金变化
