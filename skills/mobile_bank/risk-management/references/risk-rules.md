# 风险评分规则与反洗钱合规

## 风险因子评分表

每笔操作对以下因子求和，总分 > 阈值触发对应处理：

| 因子 | 分值 | 说明 |
|------|------|------|
| 首次转账至该收款账户 | +20 | 历史无转账记录 |
| 大额（单笔 ≥ 5万） | +15 | 触发大额申报提示 |
| 超大额（单笔 ≥ 20万） | +30 | 额外触发人工审核建议 |
| 非常用设备发起 | +25 | 近30天未使用的设备 |
| 异地登录后操作（<1小时） | +30 | 距上次登录距离 > 300km |
| 凌晨操作（01:00–05:00） | +10 | 异常时间段 |
| 对方账户频繁收款（>10笔/天） | +20 | 疑似洗钱通道账户 |
| 对方账户开户时间 < 7天 | +25 | 高风险新账户 |
| 近1小时内第3次以上转账 | +20 | 快速分散转账特征 |
| 本月累计转账 > 历史均值3倍 | +15 | 异常频率 |
| 备注含"投资平台"/"充值"/"刷单" | +20 | 欺诈关键词 |
| 备注含"公安"/"安全账户"/"核查" | +40 | 高度疑似诈骗话术 |
| 收款方为个人 + 金额 ≥ 5万 | +10 | 对私大额关注 |

**最高上限**：100分（多因子叠加不超过100）

## 可疑交易特征清单（AML）

根据《中国人民银行关于加强支付结算管理防范电信网络新型违法犯罪有关事项的通知》：

### 客观特征（直接判断）

1. 单笔人民币交易 ≥ 5万元（大额交易报告义务）
2. 当日累计人民币交易 ≥ 20万元
3. 当日累计外汇交易 ≥ 1万美元等值
4. 当月对同一账户累计汇入 ≥ 20笔或 ≥ 20万元
5. 账户存在与本人身份/职业/业务明显不符的资金往来

### 行为特征（需询问确认）

1. 账户短时间内被多地同时使用
2. 资金存入后立即分散转出（分拆洗钱特征）
3. 频繁小额存款后大额支出（结构性交易）
4. 收款后立即全额转出（中转账户特征）

## 欺诈话术识别关键词库

以下关键词出现在对话中须触发风险提示：

### 极高风险（分值+40）
- "安全账户"、"核查账户"、"资金保护账户"
- "配合调查"、"涉嫌洗钱"、"涉嫌诈骗"
- "公安局转账"、"检察院要求"、"法院通知"

### 高风险（分值+20）
- "刷单返利"、"轻松赚钱"、"零风险高收益"
- "帮我垫付"、"先转给我"、"我来还你"
- "投资平台充值"、"数字货币入金"

### 中风险（分值+10）
- "不要告诉家人"、"这是秘密"
- "限时优惠"、"今天必须操作"
- "虚拟货币"、"境外账户"

## 冷静期规则

| 触发条件 | 冷静期时长 | 可绕过方式 |
|---------|----------|---------|
| 风险分 61–80 | 1小时延迟 | 用户书面声明已了解风险 |
| 风险分 81–100 | 24小时延迟 | 仅限柜台办理 |
| 首次大额转账（≥10万）至新收款方 | 24小时延迟 | 电话回访确认 |
| 检测到欺诈关键词（极高风险类） | 阻断，无法绕过 | 须人工客服核实 |

## 大额交易合规要求

### 报告义务（银行自动上报，告知用户）

根据《金融机构大额交易和可疑交易报告管理办法》（2017年）：

| 类型 | 阈值 | 报告周期 |
|------|------|---------|
| 人民币现金存取 | 单笔 ≥ 5万 | T+1 上报 |
| 银行转账（对公） | 单笔 ≥ 200万 | T+1 上报 |
| 银行转账（对私） | 单笔 ≥ 5万 | T+1 上报 |
| 外汇转账 | 单笔 ≥ 1万美元 | T+1 上报 |
| 跨境汇款 | 单笔 ≥ 1000美元 | T+1 上报 |

**告知话术**：
> "根据监管规定，本笔交易属于大额交易，银行将按照法规要求进行记录上报，不影响您的正常资金使用。"

### 个人购汇年度限额

- 个人年度购汇额度：等值 **5万美元**
- 超额须向外汇局申报（非违规，但需提交申请材料）
- 用途须填写：贸易、留学、医疗、旅游等
