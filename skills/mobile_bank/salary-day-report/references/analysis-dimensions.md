# 分析维度与分类规则

| 类别 | 匹配关键词 |

| 工资 | 工资、薪资、代发、salary |
| 转账收入 | 转入、收到转账 |

| 退款 | 退款、退货 |
| 其他收入 | 以上均不匹配的入账 |

## 财务健康指标

### 储蓄率

```
储蓄率 = (总收入 - 总支出) / 总收入 × 100%
```

| 储蓄率 | 评级 | 建议 |
|--------|------|------|
| ≥ 30% | 优秀 | 继续保持，考虑增加理财配置 |
| 20%-30% | 良好 | 健康水平，可适当优化消费 |
| 10%-20% | 一般 | 审视非必要支出 |
| < 10% | 偏低 | 建议制定预算控制计划 |
| < 0% | 入不敷出 | 需立即审视支出结构 |

### 负债月供比

```
负债月供比 = (贷款月供 + 信用卡最低还款) / 月收入 × 100%
```

| 比值 | 评级 | 建议 |
|------|------|------|
| < 30% | 安全 | 负债可控 |
| 30%-50% | 关注 | 避免新增负债 |
| > 50% | 警告 | 建议还清部分债务 |

## 优化建议生成规则

按优先级触发：

1. **逾期风险** — 还款日 ≤ 5 天且未还 → 提醒立即还款
2. **高息负债** — 信用卡最低还款或分期利率 > 15% → 建议全额还款
3. **闲置资金** — 活期余额 > 月支出 × 2 → 建议配置理财
4. **消费异常** — 某类支出占比 > 40% → 提示关注
5. **收入增长** — 本月收入 > 上月 × 1.1 → 建议提升储蓄率
