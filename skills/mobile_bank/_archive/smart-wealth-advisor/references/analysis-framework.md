# 理财分析框架

## 风险等级映射

| 用户风险等级 | 适配产品风险 | 典型产品 |
|------------|------------|---------|
| 保守型 | R1 | 货币基金、国债 |
| 稳健型 | R1-R2 | 货币基金、纯债基金、银行理财 |
| 平衡型 | R1-R3 | 混合基金、指数基金 |
| 进取型 | R1-R4 | 股票基金、混合基金 |
| 激进型 | R1-R5 | 股票基金、衍生品 |

## 闲置资金识别规则

1. 取所有储蓄卡活期余额总和
2. 减去日常缓冲额（默认 ¥5,000，可根据交易明细动态调整）
3. 剩余为可理财金额

**动态缓冲调整**：若最近30天日均支出 > ¥3,000，缓冲额上调至月均支出 × 2。

调用方式：`transaction_history`（time_range=最近30天）→ 计算日均支出 → 确定缓冲额。

## 持仓诊断维度

### 收益表现评级

| 评级 | 标准 |
|------|------|
| 优 | 年化收益 ≥ 同类产品中位数 × 1.2 |
| 中 | 年化收益在同类中位数 ± 20% |
| 差 | 年化收益 < 同类中位数 × 0.8，或持续亏损 |

### 风险匹配度

- **匹配** — 产品风险等级 ≤ 用户风险等级对应最高级别
- **偏高** — 产品风险超出用户等级 1 级（提示关注）
- **严重偏高** — 产品风险超出用户等级 2 级以上（建议调整）

## 推荐排序逻辑

按以下优先级排序推荐：

1. **到期续投** — 即将到期（≤ 30天）的产品优先处理
2. **风险纠偏** — 风险偏高的持仓建议赎回转投
3. **收益优化** — 表现差的持仓建议替换为同风险更优产品
4. **闲置资金** — 将闲置资金配置到适配产品

## 推荐约束

- 单一产品占总理财资产不超过 40%
- 同风险等级产品不超过 3 只
- 必须保留活期缓冲额，不可全部投入理财
- 所有推荐附带风险提示："理财非存款，过往业绩不代表未来表现"
