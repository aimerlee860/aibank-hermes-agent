# 医学影像AI文献综述技能

一个用于撰写医学影像AI研究全面文献综述的Claude Code技能。

## 概述

此技能为撰写关于CT、MRI、X射线影像中的分割、检测和分类等主题的综述论文、系统综述和文献分析提供系统化工作流程。

## 特性

- **结构化7阶段工作流程**，用于文献综述写作
- **特定领域模板**，涵盖多个医学影像领域
- **标准化写作风格**，使用模糊语言和引用模式
- **质量检查清单**，确保完整性
- **Zotero集成**，用于参考文献管理

## 安装

将此技能文件夹复制到您的Claude Code技能目录：

```
~/.claude/skills/medical-imaging-review/
```

## 触发关键词

技能在检测到以下内容时自动激活：
- "综述论文"
- "综述"
- "文献综述"
- "综述"
- 提及撰写医学影像深度学习的学术综述

## 文件

| 文件 | 描述 |
|------|-------------|
| `SKILL.md` | 主要技能定义和快速参考 |
| `WORKFLOW.md` | 详细的7阶段工作流程指南 |
| `TEMPLATES.md` | 项目文件模板（CLAUDE.md、IMPLEMENTATION_PLAN.md） |
| `DOMAINS.md` | 特定领域的方法类别和数据集 |

## 支持的领域

- **冠状动脉分析**（CCTA）
- **肺部影像**（CT/X射线）
- **脑部影像**（MRI/CT）
- **心脏影像**（MRI/CT/超声）
- **病理学**（全切片图像）
- **视网膜影像**（眼底/OCT）
- **通用医学影像分割**

## 工作流程阶段

1. **项目初始化** - 创建项目结构和模板
2. **文献收集** - 通过Zotero/网络搜索收集论文
3. **大纲开发** - 定义章节结构
4. **章节写作** - 使用标准模板撰写每个章节
5. **表格和图形** - 创建比较表和图形占位符
6. **质量保证** - 结构、内容和语言检查
7. **增量更新** - 新文献的版本控制

## 标准综述结构

```
# [标题]：最新进展和未来方向
## 要点
## 摘要
## 1. 引言
## 2. 数据集和评估指标
## 3. 深度学习方法
## 4. 下游应用
## 5. 商业产品和临床转化
## 6. 讨论
## 7. 结论
## 参考文献
```

## 写作指南

- 使用**模糊语言**："可能"、"表明"、"似乎"
- 避免绝对声明
- 每个声明都需要引用支持
- 每个方法部分都需要一个**局限性**段落
- 每个主要部分包含**比较表**
- 目标每篇综述**80-120篇参考文献**
