# 7阶段文献综述工作流程

## 阶段1：项目初始化

创建项目结构：
```
project_root/
├── CLAUDE.md              # 写作指南
├── IMPLEMENTATION_PLAN.md # 分阶段计划
├── manuscript_draft.md    # 主要手稿
└── figures/               # 图形占位符
```

**操作：**
1. 从模板创建 `CLAUDE.md`（参见TEMPLATES.md）
2. 创建带阶段的 `IMPLEMENTATION_PLAN.md`
3. 初始化空的 `manuscript_draft.md`

## 阶段2：文献收集

### 数据来源

#### 1. ArXiv MCP（最新深度学习方法）

最适合：前沿架构、预印本、AI/ML进展

**搜索策略：**
```
查询："[主题] AND (分割 OR 检测 OR 分类)"
类别：cs.CV, eess.IV, cs.LG
日期：最近2-3年以获取最新方法
最大结果：每个查询50-100篇
```

**工作流程：**
1. 使用 `search_papers` 和主题关键词
2. 审查标题和摘要的相关性
3. 使用 `download_paper` 获取关键论文
4. 使用 `read_paper` 提取方法细节

**示例查询：**
- "医学影像分割transformer"
- "冠状动脉深度学习"
- "CT扫描神经网络"

#### 2. PubMed MCP（临床和生物医学文献）

最适合：临床验证、医学背景、同行评议研究

**搜索策略：**
- 使用MeSH术语获得精确结果
- 按出版物类型过滤（综述、临床研究）
- 专注于临床结果和验证

**示例MeSH查询：**
- "深度学习"[MeSH] AND "冠状动脉"[MeSH]
- "图像处理，计算机辅助"[MeSH] AND "X射线计算机断层扫描"[MeSH]

#### 3. Zotero（现有库和组织）

最适合：管理收集的参考文献、现有集合

**工作流程：**
1. 连接到Zotero API或使用Zotero-MCP
2. 按主题浏览现有集合
3. 导出元数据用于引用管理

### 收集工作流程

**操作：**
1. **ArXiv搜索** - 最新方法和架构（50-100篇论文）
2. **PubMed搜索** - 临床验证研究（30-50篇论文）
3. **Zotero检查** - 现有相关集合
4. **WebSearch** - 补充来源（IEEE、Springer、Google Scholar）
5. **分类** 按方法/应用分类论文
6. 创建文献矩阵：

| 类别 | 子类别 | 关键论文 | 数量 | 来源 |
|----------|-------------|------------|-------|--------|
| 方法  | CNN/U-Net   | [refs]     | N     | ArXiv  |
| 方法  | Transformer | [refs]     | N     | ArXiv  |
| 临床 | 验证  | [refs]     | N     | PubMed |
| 数据集 | 公共      | [refs]     | N     | 混合  |

7. **差距分析** - 识别缺失的主题或时间段
8. **定向搜索** - 用额外查询填补差距

## 阶段3：大纲开发

**操作：**
1. 根据文献类别定义章节标题
2. 将论文映射到章节
3. 规划比较表
4. 设计图形占位符

**输出：** IMPLEMENTATION_PLAN.md中的详细大纲

## 阶段4：章节写作

对于每个主要章节：

1. **撰写引言**（1-2段关于动机）
2. **描述方法** 使用标准模板
3. **添加性能数据** 使用一致的指标
4. **撰写局限性** 段落
5. **创建比较表**
6. **更新参考文献**

**进度跟踪：** 使用TodoWrite为每个章节

## 阶段5：表格和图形

**必需表格：**
- 表1：公共数据集
- 表2：方法比较
- 表3：商业产品（如适用）

**图形占位符：**
- 图1：综述概述/分类
- 图2：方法演进时间线
- 图3：代表性架构
- 图4：临床工作流程

## 阶段6：质量保证

**结构检查：**
- [ ] 存在要点
- [ ] 所有章节都有摘要表
- [ ] 标题层次一致

**内容检查：**
- [ ] 涵盖所有主要方法
- [ ] 讨论局限性
- [ ] 明确阐述未来方向

**语言检查：**
- [ ] 使用模糊语言
- [ ] 术语一致
- [ ] 过渡流畅

**参考文献检查：**
- [ ] 80-120篇参考文献
- [ ] 包含最新文献
- [ ] 按主题组织

## 阶段7：增量更新

当有新文献可用时：

1. **分类** 新论文
2. **更新 CLAUDE.md** 参考文献来源
3. **更新 IMPLEMENTATION_PLAN.md** 新阶段
4. **识别插入点** 在手稿中
5. **更新章节** 新方法
6. **添加新章节** 如果出现新范式
7. **更新表格** 新数据
8. **扩展参考文献**

**版本控制：**
```markdown
## 变更日志
### [日期] - v1.1
- 添加章节3.X [新类别]
- 更新表2，包含N个新方法
- 添加参考文献#XX-#YY
```
