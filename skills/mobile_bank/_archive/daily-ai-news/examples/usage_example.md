## 技能调用示例代码

### 使用Python脚本
```python
import json
from datetime import datetime

# 加载示例数据
with open('sample_news_data.json', 'r', encoding='utf-8') as f:
    news_data = json.load(f)

# 调用聚合器脚本
import subprocess
import sys

# 方法1：直接调用脚本
result = subprocess.run([
    sys.executable, 
    '/skills/daily-ai-news/scripts/ai_news_aggregator.py',
    '--input', 'sample_news_data.json',
    '--output', 'daily_briefing.json',
    '--max-age', '7',
    '--min-relevance', '3'
], capture_output=True, text=True)

print(result.stdout)

# 方法2：导入模块使用
sys.path.append('/skills/daily-ai-news/scripts')
from ai_news_aggregator import AINewsAggregator

aggregator = AINewsAggregator()
filtered_news = aggregator.filter_news(news_data)
briefing = aggregator.generate_briefing(filtered_news)

print(f"生成简报标题：{briefing['title']}")
print(f"包含新闻数：{briefing['statistics']['total_news']}")
```

### 使用task工具进行网络搜索
```python
# 使用task工具委托子agent进行网络搜索
search_tasks = []

# 并行搜索多个来源
sources = [
    "MIT Technology Review AI news",
    "TechCrunch artificial intelligence", 
    "Google AI Blog latest",
    "OpenAI Blog updates"
]

for source in sources:
    task_description = f"""
    搜索最新的AI新闻，来源：{source}
    要求：
    1. 搜索过去24小时的内容
    2. 提取新闻标题、摘要、链接、发布时间
    3. 过滤低质量内容
    4. 返回结构化JSON数据
    """
    search_tasks.append(task(task_description, subagent_type="query-specialist"))

# 等待所有任务完成并整合结果
all_news = []
for task_result in search_tasks:
    news_items = json.loads(task_result)
    all_news.extend(news_items)
```

## 最佳实践建议

1. **并行处理**：对于多个新闻来源，使用并行任务同时搜索以提高效率
2. **缓存机制**：对于频繁查询的新闻源，考虑实现缓存以减少重复请求
3. **错误处理**：网络搜索可能失败，需要完善的错误处理和重试机制
4. **用户定制**：允许用户自定义新闻源、关键词和时间范围
5. **定期更新**：对于每日简报，可以设置定时任务自动运行
6. **质量监控**：定期评估新闻源的质量，更新可信来源列表
